# Sistema de Gestión Comunitaria – Residencial Los Robles

Este proyecto es una aplicación web para la gestión comunitaria de la colonia Residencial Los Robles.
El sistema permite mejorar la comunicación entre los residentes y la mesa directiva, facilitar el pago de cuotas de mantenimiento, y centralizar información importante de la comunidad.
La aplicación surge como solución a problemas detectados en la administración de la colonia, como errores en registros de pagos, falta de comunicación efectiva y dificultad para contactar a la mesa directiva o servicios de emergencia.

## Funcionalidades principales
Gestión de usuarios
•	Registro e inicio de sesión
•	Roles de usuario:
o	Residentes
o	Administradores (mesa directiva)
Gestión de residentes
•	Registro de información de residentes
•	Dirección y datos de contacto
•	Administración de viviendas
Sistema de pagos
•	Pago de cuotas de mantenimiento en línea
•	Historial de pagos
•	Registro automático de transacciones
•	Consulta de adeudos
Sistema de notificaciones
•	Envío de alertas por:
o	Correo electrónico
o	SMS
•	Tipos de alertas:
o	Recordatorios de pago
o	Avisos de servicios
o	Alertas de emergencia
o	Comunicados oficiales
Directorio de servicios de emergencia
•	Hospitales
•	Policía
•	Bomberos
•	Protección civil
•	Información de contacto y ubicación
Información de la mesa directiva
•	Integrantes de la administración
•	Roles y responsabilidades
•	Información de contacto
•	Calendario de reuniones
